# Native Zoom Manager

Seu zoom preferido, em cada site, automaticamente.

O Native Zoom Manager é uma extensão Chrome que lembra o nível de zoom ideal para cada página que você visita. Em vez de ajustar manualmente toda vez, defina uma vez e a extensão cuida do resto — inclusive em PDFs.

## O que ela faz

- **Zoom por site**: Defina níveis diferentes para cada URL ou domínio
- **Smart Zoom**: Ajuste automático baseado na resolução da sua tela
- **Sincronização**: Suas configurações acompanham você entre dispositivos via Supabase
- **PDFs nativos**: Zoom persistente também no visualizador de PDF do Chrome
- **Controles rápidos**: Ctrl+Roda do Mouse para ajustar, Alt+Shift+R para resetar

## Instalação

1. Clone este repositório
2. Abra `chrome://extensions/` no Chrome
3. Ative o "Modo do desenvolvedor"
4. Clique em "Carregar sem compactação" e selecione a pasta `extensao/`
